Experiments

HTML5/CSS3/WebGL
  • Jezzball. An HTML5 Canvas Jezzball game, written entirely in Javascript. Highscores are uploaded to the server.
  • Snake Game. An HTML5 Canvas Snake game, written entirely in Javascript.
  • Puzzle Bobble Clone Game. An HTML5 Canvas Puzzle Bobble-like game, written entirely in Javascript.
  • Sokoban Clone Game. A simple HTML5 Canvas Sokoban-like game, written entirely in Javascript. Only has 2 levels and simple graphics.
  • Bejeweled Clone Game. A Bejeweled clone game written in Javascript. Uses CSS3 3D transforms. Works only in Chrome.
  • Connect Four Game. A game written in Javascript. Uses CSS3 3D transforms. Works only in Chrome.
  • Minesweeper. A minesweeper clone written in Javascript.
  • Peg Solitaire. A peg solitaire clone written in Javascript.
  • Sliding puzzle game. A sliding puzzle game written in Javascript.
  • Memory game. A memory game written in Javascript. Uses HTML5 FileIO API to import your own images. Works only in Chrome.
  • 3D Pong. A multiplayer 3D Pong game made with WebGL, node.js and socket.io. Does not work in IE. [ TAKEN DOWN UNTIL I CAN HOST NODE.JS AGAIN ]

F#
  • Chip-F. A Chip-8 emulator/interpreter. The first emulator I ever wrote and the project I used to learn F#. One file and few lines of code thanks to the magic of F#.
  • FunBoy. A GameBoy z80 emulator written in F# using the fewest lines of code possible while still being readable (lacks things like sound though.)

Nintendo 3DS
  • Jezzball 3DS. A 3D Jezzball homebrew game for the Nintendo 3DS.
  • Piano training for 3DS. A Piano training homebrew app for the 3DS.
